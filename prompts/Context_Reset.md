# Context Reset Prompt

Use this when resuming InterDoor work in a fresh context.

## Steps

1. Read `00_Project_Charter.md`.
2. Read `01_Project_Rules.md`.
3. Read the phase-specific prompt for the task.
4. Read `07_Backlog.md` and `08_Decisions.md`.
5. Inspect current code before editing.
6. Check `git status --short`.
7. Make only the minimal changes required for the current task.
8. Verify with tests or explain why verification was not possible.

## Reminder

InterDoor is the framework. Current games are clients and examples. Preserve existing folders unless the task explicitly targets them.
