# Quarantined Faulty Implementation

This directory preserves the removed Dominion/Empire Ascendant implementation and local build
artifacts as historical source material only.

It is not accepted Phase 1 InterDoor code, not part of the active Go module, and must not be
built, released, deployed, or used as architecture authority.

Empire Ascendant is deferred to Phase 2 or later. Any future implementation needs a fresh design
review against the real Dominion reference material and explicit approval before code is added back
to the InterDoor build surface.
