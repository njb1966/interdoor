# Packaged Legacy Design Documents

These files are packaged copies of the legacy Ledger of the Low/Dornhaven design documents.

They are useful game-specific source material, but they are not the canonical InterDoor Phase 1
framework docs. The root numbered docs define the current framework, protocol, operator, testing,
and architecture decisions.

Do not derive a general game-node framework from Ledger alone. Generic game-node guidance should
wait until a second approved game node exists and the two implementations can be compared.
