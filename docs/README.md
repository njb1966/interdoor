# Legacy Design Documents

The files in this directory are preserved source material for Ledger of the Low/Dornhaven and
early InterDoor design history.

They are not the canonical Phase 1 framework documentation. For current framework, protocol,
operator, testing, and architecture decisions, use the root numbered documents:

- `00_Project_Charter.md`
- `01_Project_Rules.md`
- `02_Architecture.md`
- `03_Hub_Specification.md`
- `04_Federation_Protocol.md`
- `05_Game_SDK.md`
- `06_Code_Modernization.md`
- `07_Backlog.md`
- `08_Decisions.md`
- `09_Testing.md`

`SYSOP_GUIDE.md` remains practical operator material, but root docs win if a framework decision
conflicts.

Do not extract a generic game-node framework from Ledger alone. Wait until Ledger of the Low and a
second approved game node, such as a properly designed Empire Ascendant, can be compared.
